# Sample Data

To test the dashboard with sample videos, you can:

1. **Download free driving videos from:**
   - Pexels: https://www.pexels.com/search/videos/driving/
   - Pixabay: https://pixabay.com/videos/search/driving/
   - Unsplash: https://unsplash.com/s/videos/driving

2. **Or create test videos:**
   - Use your phone to record short driving clips
   - Screen record driving simulation games
   - Use dashcam footage if available

3. **Sample video names for testing:**
   - `normal_highway_drive.mp4`
   - `aggressive_lane_change.mp4`
   - `distracted_phone_use.mp4`
   - `drowsy_driving_signs.mp4`
   - `speeding_violation.mp4`

The dashboard will automatically generate analysis results for any uploaded video.
